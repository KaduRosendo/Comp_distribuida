# Sistema de Cadastro de Produtos em C

Atividade prática sobre gerenciamento de memória em C: alocação dinâmica, ponteiros, passagem por referência e vetores dinâmicos.

## Como compilar

```bash
gcc -Wall -o atividade main.c
```

## Como rodar

```bash
./atividade
```

> No Windows:
> ```bash
> .\atividade.exe
> ```

## Funcionalidades

- Adicionar produto
- Listar todos os produtos com valor total do estoque
- Buscar produto por código
- Atualizar estoque
- Remover produto
- Liberação de memória ao encerrar

## Requisitos

- GCC instalado
